Chat with another salesmate CRM person easily.

== Description ==

This plugin allows you to chat messenger with another salesmate CRM user.

Features:

*   Chat with other users easily.
*   Need to provide only basic details like Workspace ID, Tenant ID and API key.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload .zip file from "Add New" at admin site in "Plugin" menu OR put unzip folder directly at location "/wp-content/plugins"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Salesmate Messenger and provide required configuration from your Salesmate CRM Account
